# DevOps Homelab
This repository is my DevOps & Cloud learning environment.

## Goal
Build a homelab for hands-on practice with:
- Linux system administration
- Networking and SSH
- Docker and containerization
- Kubernetes (basic cluster setup)
- Automation (Bash, Python scripting)
- CI/CD (GitHub Actions, Jenkins, GitLab CI/CD)
- Infrastructure as Code (Terraform, Ansible)
- Cloud fundamentals (AWS, Azure, GCP)

## Roadmap
1. Linux + networking
2. Docker
3. Automation
4. CI/CD
5. Kubernetes
6. IaC + Cloud

## Structure
- virtualbox/   - VM configuration and setup
- scripts/      - automation scripts (Bash, Python)
- apps/         - applications and services
